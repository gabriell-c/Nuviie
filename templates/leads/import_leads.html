Como usar a extensão

  1. Chrome → Extensões → Modo desenvolvedor → Carregar sem compactação → pasta Nuviie/extension
  2. Abra o Google Maps, busque (ex: advogado Ribeirão Preto), role a lista
  3. Clique na extensão → Extrair completo
  4. Exporte JSON/CSV ou envie direto ao Nuviie com o token do .env
